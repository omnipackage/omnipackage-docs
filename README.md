# OmniPackage documentation

Created with [mkdocs](https://www.mkdocs.org) and [mkdocs-material](https://squidfunk.github.io/mkdocs-material)

Install dependencies:

```shell
make deps
```

Run with live reload:

```shell
make serve
```

Build static site (output in `site/`):

```shell
make build
```
